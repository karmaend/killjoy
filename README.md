Killjoy is a curated cybersecurity information hub designed to simplify complex security concepts and provide structured knowledge for learners, enthusiasts, and aspiring professionals.

https://karmaend.github.io/killjoy/

# Features
📚 Well-organized cybersecurity topics
🧠 Concept-driven explanations
🌍 Static website for easy access
⚡ Lightweight and fast
🎯 Beginner to intermediate friendly

# Tech Stack
1. HTML5
2. CSS3
3. JavaScript
4. GitHub Pages (for deployment)

# To run locally:
git clone https://github.com/karmaend/killjoy.git
cd killjoy

index.html
