Killjoy is a curated cybersecurity information hub designed to simplify complex security concepts and provide structured knowledge for learners, enthusiasts, and aspiring professionals.

https://karmaend.github.io/killjoy/

# Features
1. Well-organized cybersecurity topics
2. Concept-driven explanations
3. Static website for easy access
4. Lightweight and fast
5. Beginner to intermediate friendly

# Tech Stack
1. HTML5
2. CSS3
3. JavaScript
4. GitHub Pages (for deployment)

# To run locally:
1. git clone https://github.com/karmaend/killjoy.git
2. cd killjoy

# Open in browser:
index.html
